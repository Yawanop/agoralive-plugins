# Exemples

Les gabarits de référence dans `assets/` sont eux-mêmes les exemples travaillés
complets — l'article DentalMonitoring validé avec Paul, dans les deux formats :

- `assets/template-web.html` — article sponsorisé complet, format web interactif
  (mise en page 3 colonnes, onde ambiante, apparition au défilement).
- `assets/template-print.html` — le même article, édition imprimable A4.

Pour créer un nouvel article, partir de ces fichiers et les adapter selon le
mode visé : voir `references/mode-conference.md` ou `references/mode-sponsorise.md`.
