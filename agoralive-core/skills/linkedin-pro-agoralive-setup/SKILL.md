# linkedin-pro-agoralive-setup — Setup page LinkedIn entreprise

## Mission

Mission 🔴 actuelle d'Éloïse : créer la page LinkedIn entreprise AgoraLive — visibilité B2B + attraction candidats. Éloi structure le setup complet.

---

## Procédure

### Étape 1 — Vérifie les prérequis

Pour créer une page LinkedIn entreprise :
- Identité visuelle finalisée ou v0 minimum (cf. `identite-agoralive`)
- Domaine email pro (✅ @agoralive.ai)
- Au moins 1 employé déjà sur LinkedIn pour rattachement

Si l'identité visuelle n'est pas prête → propose de créer la page avec branding minimaliste, à enrichir après.

### Étape 2 — Drafte la fiche entreprise

```
📛 Nom : AgoraLive
🌐 Site web : agoralive.ai
🏢 Secteur : Médias en ligne (Online Media) · ou Édition spécialisée
👥 Taille : 2-10 employés
📅 Année de création : <année>
🏛️ Type : Société privée
📍 Siège : Paris ou Reims (selon choix)
```

### Étape 3 — Drafte la description (À propos)

Format LinkedIn : punchy, 2-3 paragraphes, 2000 caractères max.

Template :
```
[Hook accroche - 1 ligne forte sur la mission]

AgoraLive transforme l'éphémère en patrimoine. Nous captons les congrès médicaux, scientifiques et professionnels, puis en produisons des articles éditorialisés et un journal périodique — pour que la connaissance partagée en salle de conférence vive au-delà du présentiel.

🎯 Notre approche
• Captation audio professionnelle (intervenants, panels, tables rondes)
• Production éditoriale (transcription, structuration, articles SFODF-grade)
• Distribution multi-canale (numéro digital + papier, articles autonomes)

🤝 Notre différenciation
[USP unique vs concurrents - 1-2 lignes]

💡 Pourquoi nous suivre ?
• Pour découvrir les coulisses du journalisme scientifique de demain
• Pour voir les meilleures interventions des congrès dentaires, médicaux, et bientôt cardio, dermato, etc.
• Pour les contenus exclusifs sur l'IA appliquée à l'édition scientifique
```

(Cette description sera affinée selon l'étude 4P + identité visuelle finalisée.)

### Étape 4 — Drafte les premiers posts (5-10)

Pour amorcer la page, programmer 5-10 posts variés :

1. **Post de lancement** : "AgoraLive arrive — qui sommes-nous, pourquoi" + équipe
2. **Post mission** : la vision éditoriale d'AgoraLive
3. **Post métier** : tour d'horizon des congrès dentaires (référence SFODF)
4. **Post coulisses** : un jour avec un captationnaire AgoraLive
5. **Post tech** : comment l'IA aide à structurer un article scientifique
6. **Post équipe** : présentation d'un membre (rotation)
7. **Post client** (si premier signé) : "merci à <client>, fier de capter votre congrès"
8. **Post pédagogie** : "comment lire un article de revue scientifique"
9. **Post anniversaire** : "1 mois d'AgoraLive, voici ce qu'on a appris"
10. **Post recrutement** : ouverture poste

Pour chaque post, drafter via `linkedin-post-eloise` (skill à créer si pas dispo, ou via Éloi qui le bricolera).

### Étape 5 — Coordonne avec profils perso

Demande à chaque membre de :
1. Mettre à jour son profil LinkedIn perso (titre + société = AgoraLive)
2. Se rattacher comme employé de la nouvelle page
3. Partager / liker le post de lancement pour amorcer l'audience

### Étape 6 — Stratégie de croissance

Drafte une stratégie 90 jours :
- **M+1** : 50 followers (équipe + premier cercle perso)
- **M+2** : 200 followers (sponsors, candidats, écosystème dentaire)
- **M+3** : 500 followers (résultat d'une publication régulière 2x/semaine)

### Étape 7 — Restitue le récap

```
📣 Setup page LinkedIn AgoraLive — checklist
☑️ Page entreprise créée : <URL>
☑️ Description à propos : <draftée>
☑️ Branding : <logo / header — ou v0 minimaliste>
☑️ 5-10 premiers posts draftés (calendrier programmé)
☑️ Profils perso équipe rattachés
🎯 Cible 90 jours : 500 followers

👉 Prochaines actions :
1. Éloïse : finaliser branding visuel
2. Tous : rattacher profil perso
3. Éloi : publier post lancement + 1 post / 3 jours
```

---

## Anti-patterns

- ❌ **Ne lance pas la page sans identité visuelle minimum** — la première impression compte.
- ❌ **Ne fais pas de posts génériques marketing creux** — AgoraLive doit avoir un POV éditorial.
- ❌ **N'ajoute pas tous les membres comme admin** — 2 admins (Éloïse + 1 backup) suffisent.
- ❌ **Ne sur-promette pas dans l'à propos** — reste sobre, authentique.
